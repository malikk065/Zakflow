Firebase-Projekt erstellen

Schritt 1 von 5

  1. Öffne die Firebase Console
  2. Klicke auf "Projekt hinzufügen" (Add project)
  3. Gib einen Namen ein (z.B. "Rechnungsapp")
  4. Google Analytics: kannst du deaktivieren (nicht nötig)
  5. Klicke "Projekt erstellen" und warte 30 Sekunden

Datenbank aktivieren

Schritt 2 von 5 – Firestore

  1. Im linken Menü: Build → Firestore Database
  2. Klicke "Datenbank erstellen" (Create database)
  3. Standort wählen: europe-west3 (Frankfurt)
  4. Modus: "Im Testmodus starten" (Test mode)
  5. Klicke "Aktivieren"

💡 Testmodus reicht für 30 Tage. Danach kannst du die Regeln anpassen.

Login aktivieren

Schritt 3 von 5 – Authentication

  1. Im linken Menü: Build → Authentication
  2. Klicke "Jetzt starten" (Get started)
  3. Tab "Sign-in method" öffnen
  4. Klicke "E-Mail/Passwort"
  5. Aktiviere den Schalter ganz oben → Speichern

⚠️ Wichtig! Ohne diesen Schritt funktioniert der Login nicht.

Web-App registrieren

Schritt 4 von 5 – Config holen

  1. Klicke das ⚙️ Zahnrad (oben links) → "Projekteinstellungen"
  2. Scrolle zu "Meine Apps"
  3. Klicke das </> Web-Icon
  4. App-Name: z.B. "Rechnungsapp" → "App registrieren"
  5. Du siehst jetzt firebaseConfig mit Werten – die brauchen wir gleich!
